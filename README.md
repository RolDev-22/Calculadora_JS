Este proyecto consiste en una calculadora funcional desarrollada con JavaScript puro, orientada a fortalecer habilidades iniciales en desarrollo web. La aplicación implementa:
- Captura dinámica de datos del usuario mediante eventos del navegador (click, input), sin frameworks externos.
- Manipulación directa del DOM para actualizar la interfaz en tiempo real, utilizando selectores, eventos y modificaciones de elementos HTML.
- Lógica aritmética estructurada, que permite realizar operaciones básicas como suma, resta, multiplicación y división, con control de errores y validaciones simples.
- Interfaz estilizada con CSS, diseñada para ofrecer una experiencia visual clara, responsiva y amigable.
Este proyecto sirve como base para comprender la interacción entre HTML, CSS y JavaScript, y representa un ejercicio práctico en programación imperativa, control de flujo, y diseño modular de funciones.
